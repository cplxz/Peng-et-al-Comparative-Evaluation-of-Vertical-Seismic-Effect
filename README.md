# Peng-et-al-Comparative-Evaluation-of-Vertical-Seismic-Effect

This repository contains the data, analysis scripts, and replication code for the paper: 

Peng, C. et al. (2026). "Comparative Evaluation of Vertical Seismic Effect Methods in ASCE 7-22 for the Contiguous United States." Earthquake Spectra, EERI (Accepted).
---

## Overview

This project provides a comprehensive, spatial evaluation of the vertical seismic effect design methods introduced or updated in ASCE 7-22 across the continental United States. This code processes seismic data to map, analyze, and compare the engineering implications of these regulatory design frameworks.
